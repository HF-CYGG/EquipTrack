# **App Name**: EquipTrack

## Core Features:

- Record Loan: 记录物资的借出详情，包括借用人、物资、借用时间等。
- Record Return: 记录物资的归还信息，支持拍照（非必要），并自动记录实际归还时间。
- Real-time Status View: 实时显示物资的借用状态（已借出、可用、逾期）。
- Customizable Item: 允许用户添加或自定义物资信息字段。
- Customizable Form: 支持用户自定义借用申请表单的字段 (姓名、联系电话、借用时间、预归还时间)。
- Real-time Cloud Sync: 云端实时同步物资借用状态。
- Custom Server Address: 为云端服务器预留自定义服务器地址，以便后期自行添加服务器
- User Registration: 用户可以注册为用户
- Borrowing Form: 每次借用需要填写借用表格 (姓名、联系电话、借用时间、预归还时间)

## Style Guidelines:

- 主色调: 蓝绿色 (#008080)，营造专业、可靠的感觉，体现信任感和效率。
- 背景色: 淡蓝绿色 (#E0F8F8)，主色调的淡化版本，创造干净、不突兀的背景。
- 强调色: 亮绿色 (#BFFF00)，与主色调相似，但亮度更高，用于突出交互元素。
- 字体: 'Inter'，一种无衬线字体，提供现代、易读、中性的设计，适用于标题和正文。
- 使用简单、清晰的图标来表示物资类别和操作，帮助用户快速识别和导航。
- 保持干净、有条理的布局，易于导航和理解。借用状态应清晰呈现，方便用户快速查看相关信息。
- 在更新状态或加载新数据时，加入细微的动画或过渡效果，增强用户参与感和体验。
